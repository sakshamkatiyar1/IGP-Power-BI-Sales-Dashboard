# IGP E-Commerce Sales & Profit Dashboard

An interactive **Power BI analytics project** that tracks the sales performance, profit trends, and operational insights for IGP E-Commerce across months, quarters, product categories, states, and payment modes.

## Project Description

An interactive Power BI dashboard designed for **IGP E-Commerce** to visualize and track key business metrics such as sales, profit, loss, and regional performance over time.

## Purpose

This dashboard helps stakeholders quickly understand business performance, identify trends in sales and profit, and make informed decisions to boost revenue and optimize strategies.

## Tech Stack

- **Power BI Desktop** – Core tool for creating interactive dashboards and reports  
- **Power Query** – For data extraction, transformation, and loading (ETL)  
- **Data Modeling** – Designing relationships, hierarchies, and optimized data schema  
- **DAX (Data Analysis Expressions)** – For advanced calculations and custom measures  
- **CSV** – Source data format  

## Data Source

The dashboard uses CSV files exported from **IGP E-Commerce’s internal systems**. The main data file (`data/ecommerce_details.csv` and `data/ecommerce_order.csv`) contains order and sales information which is loaded into Power BI using Power Query, cleaned, transformed, and modeled for analysis.

## Features & Highlights

### Core Dashboard Features
- **Interactive Sales Visualizations** across months, quarters, and years
- **Profit & Loss Analysis** to track profitability trends
- **State-Wise Performance** insights for regional comparison
- **Category & Sub-Category Breakdown** to understand product performance
- **Payment Mode Insights** by payment type
- **Customer Insights** showing top contributors
- **Trend Analysis with Time Filters** using dynamic slicers

### Highlights
- Supports data-driven decision making
- Clean data preparation with Power Query
- Scalable and flexible for future needs
- Uses CSV exports for easy loading and updating
